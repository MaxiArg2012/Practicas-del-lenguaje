# Prácticas del Lenguaje

Aplicación de escritorio desarrollada con Electron para un trabajo práctico de la materia Prácticas del Lenguaje.

## Estado actual

- Proyecto inicial creado con Node.js.
- Electron instalado.
- Aplicación mínima funcionando en macOS.
- Repositorio GitHub configurado con la cuenta de Maximo.

## Tecnologías utilizadas

- Electron
- Node.js
- HTML
- JavaScript
- Git / GitHub

## Objetivo

Construir una aplicación de escritorio que pueda ejecutarse inicialmente en macOS y luego ser transportada a Windows.
